# Outdoor Balkans

Outdoor directory for hunting, fishing, and hiking across the Balkans.

A scalable, SEO-first geo-location platform built with:
- Next.js (App Router)
- Supabase (Postgres + PostGIS for geo-queries)
- Leaflet + OpenStreetMap
- Deployed on Vercel

## Features (MVP focus)
- Advanced search & filters by country, region, activity, permit, fish type
- Location cards with images, short descriptions, best season, permit info
- Country-specific pages (e.g., /serbia/hiking)
- Content-driven organic growth + future monetization (premium listings, affiliates)
